# How to  Run

## Backend

  1. Install [MySQL] 
  2. Configure datasource in `application.properties`.
  3. `cd ecommercebackend`.
  4. Run `mvn install`.
  5. Run `mvn spring-boot:run`.
  6. The backend server is running on [localhost:8080]().

## Frontend
  1. Install
  2. `cd EcommerceFront`.
  3. Run `npm install`.
  4. Run `ng serve`
  5. The frontend client is running on [localhost:4200]().
